# VC-Arkanoid

PyGame으로 만든 간단한 벽돌깨기(Arkanoid 스타일) 게임입니다. 패들을 좌우로 움직여 공을 튕기고, 벽돌을 모두 제거해 스테이지를 클리어하세요. 일부 벽돌은 파워업(캡슐)을 드롭합니다.

## 요구 사항
- Python 3.8 이상
- pip (패키지 관리자)
- pygame 2.x

설치된 Python 버전을 먼저 확인하세요.

```
python --version
```

## 설치
가상환경은 선택 사항이지만 권장됩니다.

```
python -m venv .venv
.\.venv\Scripts\activate
pip install --upgrade pip
pip install pygame
```

## 실행

```
python main.py
```

- 해상도: 800x600 고정
- 첫 실행 시 바로 게임이 시작됩니다.

## 조작법
- 패들 이동: 마우스 좌우 (권장) 또는 방향키 좌/우
- 발사: 마우스 왼쪽 버튼
  - 레이저 파워업 활성 시 레이저 발사(패들 양끝에서 2발)
  - Catch 파워업으로 공을 붙잡은 상태에서 클릭하면 공을 재발사
- 일시정지: `P` 또는 `Esc`
- 재시작: 게임 오버/클리어 화면에서 `R`
- 종료: `Esc`

## 게임 규칙 요약
- 목표: 공을 떨어뜨리지 않고 벽돌을 모두 제거하여 스테이지 클리어
- 목숨: 3개. 공이 화면 아래로 떨어지면 1 감소, 0이 되면 게임 오버
- 점수: 벽돌 파괴 +10, 적 제거 +50, 보스 타격 +100
- 보스: 특정 스테이지에서 등장(DOH). 일정 주기로 폭탄을 떨어뜨립니다.

## 파워업(캡슐)
벽돌 파괴 시 확률적으로 드롭되며, 패들이 획득하면 즉시 발동합니다.

- S (Slow): 공의 속도를 일시 감소
- E (Enlarge): 패들 길이를 일시 증가
- L (Laser): 마우스 클릭으로 레이저 발사 가능
- C (Catch): 공이 패들에 닿으면 붙잡기, 클릭 시 재발사
- D (Disrupt): 공이 여러 개로 분열(멀티볼)
- B (Break): 다음 스테이지로 이동하는 출구 생성(즉시 클리어 성격)
- P (Player): 목숨 +1

일부 파워업은 일정 시간이 지나면 자동으로 해제됩니다.

## 레벨과 맵 편집
레벨 데이터는 `levels.py`의 `LEVELS` 리스트로 정의되어 있습니다. 문자 의미:

- `n`: 일반 벽돌(1회 타격 파괴)
- `s`: 단단한 벽돌(2회 타격 파괴)
- `g`: 금색 벽돌(파괴 불가, 장애물)
- 공백(` `): 빈 칸

예시:

```
LEVELS = [
    [
        "gggggggggg",
        "ssssssssss",
        "nnnnnnnnnn"
    ]
]
```

## 사운드 안내
`main.py`는 다음 경로의 사운드를 로드합니다. 파일이 없으면 무음으로 동작할 수 있습니다.

- `sounds/brick_destroy.wav`
- `sounds/powerup.wav`
- `sounds/bounce.wav`

참고: 일부 환경에서는 오디오 장치가 없을 경우 `pygame.mixer.init()`가 실패할 수 있습니다.

## 문제 해결(Troubleshooting)
- pygame 미설치 오류: `pip install pygame`으로 설치합니다.
- 오디오 장치/드라이버 오류로 실행 실패:
  - Windows CMD/Powershell에서 임시로 무음 드라이버 사용:
    - `set SDL_AUDIODRIVER=dummy` (CMD)
    - `$env:SDL_AUDIODRIVER = "dummy"` (Powershell)
  - 혹은 `sounds` 폴더에 위 사운드 파일을 추가하세요.
- 창이 너무 느린 경우: 다른 앱을 종료하고, 창을 비활성화/최소화하지 말고 실행해 보세요.
